<h3 align = "left">Quick start:</h3>

1. Copy all folders in the `libdeps` directory to the `~/Arduino/libraries` directory, and put them in `"Documents/Arduino/libraries"` for Windons users
2. Open the corresponding example
3. Choose `TTGO T-Beam` for the board
4. Select the port of the board in the port
5. The default version is 1.0/1.1. If the purchased version is v0.7, please comment out `T_BEAM_V10` in the example `utilities.h`, and uncomment `T_BEAM_V07`, if you donâ€™t know the version, please ask customer service, or Check the silkscreen on the board
6. Upload
